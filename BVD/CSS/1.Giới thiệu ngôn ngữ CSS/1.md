- CSS ngôn ngữ tách biệt với html 
- một số khái niệm trong css sẽ có cùng tên với 1 số khái niệm trong html.
ví dụ trong html để tạo màu nền thì sử dụng thuộc tính bgcolor còn trong css thì background-color
- cấu trúc của css
bộ chọn {
    thuộc tính 1:giá trị;
    thuộc tính 2:giá trị;
}

- ưu và nhược điểm của các cách thiết lập css khác nhau
+ css ngoài: phần tệp html riêng và phần tệp css riêng nên ta có thể sử dụng lại nó được ở nhiều tệp html khác nhau miễn là trong tệp html có liên kết tới tệp css đó
nhược điểm sẽ là trình duyệt thấy phần tử link hay phần tử style để thêm tệp css vào thì trình duyệt sẽ gửi một yêu cầu để lấy nội dung của tệp đó.
+ css trong: cái này không cần tạo tệp css ngoài ==> nó sẽ áp dụng cho tất cả phần tử trong trang web.
+ css nội tuyến: nó sẽ có độ ưu tiên cao nhất nhưng nó lại chỉ áp dụng cho 1 phần tử nhất định.
