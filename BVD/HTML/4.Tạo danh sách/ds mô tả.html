HTML là gì?
HTML (Hypertext Markup Language) hay còn được gọi là ngôn ngữ đánh dấu siêu văn bản được thiết kế để tạo nên các trang web trên World Wide Web (www). HTML có chức năng giúp người dùng có thể định dạng, thiết kế cấu trúc các thành phần của một trang web hay ứng dụng, bao gồm tiêu đề (headings), liên kết (links), bảng (table), hình ảnh, đoạn văn bản và nhiều yếu tố khác.
CSS là gì?
CSS là viết tắt của Cascading Style Sheets một ngôn ngữ định dạng dùng để mô tả cách trình bày các phần tử HTML trên một trang web. CSS được sử dụng để kiểm soát giao diện của trang web, bao gồm phông chữ, màu sắc, khoảnh cách, bố cục và các hiệu ứng hình ảnh. Tóm lại, CSS là ngôn ngữ giúp bạn “trang trí” cho trang web của mình, làm cho nó trở nên thẩm mỹ, hấp dẫn, dễ nhìn hơn.
